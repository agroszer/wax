CanCut(self)
Return true if text is selected and can be cut.

Return true if editing should succeed.

CanPaste(self)
Return true if a paste should succeed.

Copy(self, data, command, ps2, ps1)
Copy selection and place it on the clipboard.

CopyWithPrompts(self, data, command)
Copy selection, including prompts, and place it on the clipboard.

CopyWithPromptsPrefixed(self, data, spaces, command)
Copy selection, including prompts prefixed with four spaces, and place it on the clipboard.

Cut(self)
Remove selection and place it on the clipboard.

OnChar(self, event, currpos, stoppos, command, key)
Keypress event handler. Only receives an event if OnKeyDown calls event.Skip() for the corresponding event.

OnHistoryInsert(self, step, endpos, startpos)
Insert the previous/next command from the history buffer.

OnHistoryReplace(self, step)
Replace with the previous/next command from the history buffer.

OnHistorySearch(self, searchText, i, command, endpos, searchOrder, numCharsAfterCursor, startpos)
Search up the history buffer for the text in front of the cursor.

OnIdle(self, event)
Free the CPU to do other things.

OnKeyDown(self, event, shiftDown, controlDown, selecting, endpos, key, currpos, home, altDown)
Key down event handler.

Paste(self, command, ps2, data)
Replace selection with clipboard contents.

PasteAndRun(self, commands, text, lines, command, endpos, ps2, startpos, ps1, data, line)
Replace selection with clipboard contents, run commands.

redirectStdout(self, redirect)

run(self, command, prompt, verbose, endpos)
Execute command as if it was typed in directly. >>> shell.run('print "this"') >>> print "this" this >>>

runfile(self, filename, command, file)
Execute all commands in file as if they were typed into the shell.

wrap(self, wrap)

write(self, text)

writeErr(self, text)

writeOut(self, text)

zoom(self, points)
Set the zoom level. This number of points is added to the size of all fonts. It may be positive to magnify or negative to reduce.
